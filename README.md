## Déploiement GitHub Pages
- Settings → Pages → Deploy from a branch → main → / (root)

## Mettre à jour ton site (simple)
Option 1 (facile) :
1. Ouvre ton repo GitHub
2. **Add file → Upload files**
3. Glisse les fichiers mis à jour (par exemple `index.html` + `assets/`)
4. Commit changes
5. GitHub Pages se met à jour tout seul en 1-2 minutes

Option 2 (pro) :
- Utilise Git + `git push`

## Notes
- GitHub Pages ne supporte pas PHP : `forms/contact.php` fonctionne seulement sur un hébergement PHP.
- Le CV PDF est dans `assets/docs/`.
